## ⚙️ How It Works Behind the Scenes

```
User Uploads Video
    ↓
VideoUpload Component
    ├─ URL mode: Saves to database
    └─ File mode: Uploads to Supabase Storage
    ↓
Video URL stored in courses.video_url
    ↓
Dashboard refreshes
    ↓
User sees video in preview
    ↓
Student can watch video
```
